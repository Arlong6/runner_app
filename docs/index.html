📖 使用教學 —— 怎麼用本站 + 如何匯入手機/電腦日曆(點我展開)

📱 手機:加單場(每張卡片兩個鈕)

⚡ 一鍵加 Google —— 點一下直接跳進 Google 日曆、按儲存即可。手機真‧一鍵最快,但只加比賽日、沒有自訂提醒。

⏰ 含提醒 —— 同時加「比賽日 + 報名截止前提醒」(截止前幾天可在上方調)。手機會跳分享選單選行事曆、電腦則下載 .ics,步驟多一點點但不會錯過報名。

① 想一次加很多場(含報名截止提醒)

  1. 用下方搜尋或篩選(只看台灣、只看開放報名)找到想跑的賽事。
  2. 勾選賽事,可以複選(預設已隱藏「已截止報名」的場次)。
  3. 設定「報名截止前幾天提醒」(預設 7 天)。
  4. 按底部按鈕:手機會跳出分享選單 → 選「行事曆」直接加入;電腦則下載 my-races.ics。

② 電腦下載的檔案怎麼匯入 Google 日曆

用電腦版:

  1. 電腦開啟 Google 日曆網頁(calendar.google.com)。
  2. 右上角 齒輪 ⚙️ → 設定。
  3. 左側選 「匯入與匯出」。
  4. 匯入 → 選剛下載的 my-races.ics。
  5. 選擇要匯入哪個日曆 → 按 「匯入」。
✅ 完成!所有勾選的賽事會出現在日曆,報名截止日也會有 ⏰ 報名截止 事件,並在截止前提醒你。手機上的 Google 日曆 App 會自動同步顯示,不用另外操作。

③ iPhone / Apple 行事曆

  1. 把 my-races.ics 用 AirDrop 或 Email 傳到手機。
  2. 點開檔案 → 選「加入全部」到行事曆即可。

④ 賽事更新了怎麼辦

下載的 .ics 是當下的快照,之後有新賽事或你想改清單,回來重新勾選、再下載匯入一次就好(本站賽事資料每天自動更新)。重複匯入不會產生重複事件,放心。
